# SelfKey Certifiers Demo

### Demo UI/UX for Certifiers on the SelfKey Identity Network

## Overview

An app to demo proposed user flows and interfaces for the certifier role in the SelfKey Identity Network.  

## Usage

`npm i`
`npm run dev`
